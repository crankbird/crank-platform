## Summary
Tulving’s 1972 distinction between **episodic** and **semantic** memory began as a conceptual model. Subsequent neuroimaging confirmed distinct neural networks support these systems. Episodic memory involves the **hippocampus, medial prefrontal cortex, and posterior cingulate**; semantic memory involves **anterior temporal and lateral prefrontal** regions.

## Evidence Type
Neuroimaging, lesion studies

## Key Sources
- Tulving et al. (1994). *Hemispheric encoding/retrieval asymmetry in episodic memory.* **PNAS**, 91(6), 2016–2020. [DOI:10.1073/pnas.91.6.2016](https://doi.org/10.1073/pnas.91.6.2016)

## Reflection
These studies give empirical weight to Tulving’s conceptual distinction. In **Brand Engineering**, this mirrors how *episodic interactions* (user experience) and *semantic knowledge* (brand meaning) operate in parallel yet integrated loops.
